![Task tracker](https://github.com/PolinaZaprudskaya/ToDo-Angular/blob/master/tech-test/demo.gif)

## Quick start

#### clone the repo
#### change into the repo directory
cd tech-test
#### install
`npm install`
#### serve
`npm run server`
`npm run start`
#### Running unit tests
`npm run test`
